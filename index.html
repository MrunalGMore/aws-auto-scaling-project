About This Project

This web application is deployed on an AWS EC2 instance with Auto Scaling and CloudWatch monitoring enabled. It demonstrates cloud resource management, uptime optimization, and scalability.

Key Features

Auto Scaling

The app scales automatically based on traffic demand, ensuring optimal performance at all times.

CloudWatch Monitoring

Comprehensive monitoring with CloudWatch provides insights into system performance and resource usage.

Cost-Effective

By leveraging AWS features like Spot Instances, the app reduces operational costs while maintaining high availability.
